## Quick Start

[Quick Start](https://angular.io/quick-start)

`npm init @angular myApp`

`npm start`

default URL is `http://localhost:4200`

`ng new my-first-project`
`cd my-first-project`
`ng serve`

`ng generate component x`
`ng generate interface x`
